# ice-cream-sales-prediction-linear-regression
A Machine Learning project that predicts ice cream sales based on temperature using Linear Regression with Python and Scikit-learn.
## 📌 Project Overview

This project predicts ice cream sales based on temperature using the Linear Regression algorithm. The objective is to build a machine learning model that learns the relationship between temperature and ice cream sales and makes accurate predictions.

---

## 🎯 Objective

To predict ice cream sales using temperature as the input feature.

---

## 📂 Dataset

The dataset contains two columns:

- Temperature_C (Feature)
- IceCream_Sales (Target)

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 📋 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Feature Selection
6. Train-Test Split
7. Build Linear Regression Model
8. Train the Model
9. Make Predictions
10. Evaluate the Model
11. Predict New Data

---

## 📊 Model Performance

- Mean Absolute Error (MAE): **15.66**
- Mean Squared Error (MSE): **384.42**
- R² Score: **99.41%**

---

## 📈 Result

The model successfully learned the relationship between temperature and ice cream sales. It achieved an R² score of **99.41%**, indicating excellent predictive performance.
